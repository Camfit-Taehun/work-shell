# Handoff - Task Handoff Document

Generate a comprehensive handoff document for transferring work to another person.

## Parse Arguments

`$ARGUMENTS`:
- Empty - Generate handoff for current work
- `--to <person>` - Specify recipient
- `--format <md|notion>` - Output format

## 1. Gather Information

Collect from:
- Current session state
- Recent session logs
- work-shell.md context
- Git history
- Pending todos
- Decisions log

## 2. Generate Handoff Document

```markdown
# Handoff Document

**작성자**: {author}
**수신자**: {recipient}
**날짜**: {date}
**프로젝트**: {project}

---

## 현재 상태

**Branch**: `{branch}`
**Focus**: {current_focus}

### 진행 상황
{progress_summary}

### 최근 변경사항
- {recent_changes}

---

## 주요 결정 사항

| 날짜 | 결정 | 이유 |
|------|------|------|
{decisions}

---

## 미완료 작업

### 높은 우선순위
- [ ] {high_priority_tasks}

### 일반
- [ ] {normal_tasks}

---

## 알아야 할 사항

### 주의점
{warnings_and_gotchas}

### 유용한 정보
{helpful_info}

---

## 관련 파일

### 주요 파일
{key_files}

### 최근 수정 파일
{recently_modified}

---

## 다음 단계

1. {next_step_1}
2. {next_step_2}
3. {next_step_3}

---

*Generated by work-shell*
```

## 3. Output

```
## Handoff 문서 생성됨

**수신자**: {recipient or "미지정"}
**파일**: .work-shell/handoffs/{date}-{branch}.md

### 포함된 내용
- 현재 상태 요약
- 결정 사항 {N}개
- 미완료 작업 {N}개
- 주요 파일 목록

---

**Tip**: 문서를 검토하고 필요시 수정하세요.
`/share` 로 팀에 공유할 수 있습니다.
```

User argument: $ARGUMENTS

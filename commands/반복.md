# 반복 명령어

`/loop` 명령어의 한국어 버전입니다.

## 사용법

```bash
# 2시간 동안 5분마다 빌드
/반복 "빌드" "2시간" "5분"

# 10회 테스트
/반복 "테스트" "10회"

# 1시간 동안 린트 수정
/반복 "린트 에러 수정" "1시간"
```

## 기간 형식

- `30분`, `30m` = 30분
- `2시간`, `2h` = 2시간
- `1일`, `1d` = 1일
- `10회`, `10x` = 10번 반복

**자세한 내용은 `/loop` 명령어 문서를 참조하세요.**

이 명령어는 `/loop`와 동일한 방식으로 실행됩니다.
